# Multivariate Energy Analysis – Steel Industry

This project performs a full multivariate analysis of steel industry energy consumption and power quality behaviour. The analysis includes:

- Principal Component Analysis (PCA)
- K-means clustering (K=3 optimal solution)
- Linear Discriminant Analysis (LDA)
- Quadratic Discriminant Analysis (QDA)
- Canonical Correlation Analysis (CCA)
- Redundancy Analysis (RDA)
- Predictive modelling using GLMSELECT

The goal is to understand operational behaviour, classify plant load types, and forecast electricity usage for cost optimisation.
